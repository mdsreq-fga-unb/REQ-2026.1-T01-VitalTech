# Dailys - Sprint 1

Este documento centraliza o histórico das Daily Scrums (adaptação Scrum XP). Como a equipe optou por **dailys assíncronas via WhatsApp**, este espaço serve para oficializar semanalmente (ou ao fim da sprint) essas interações de desenvolvimento.

## Template de Resposta da Daily
Copie o template abaixo para registrar o status de cada membro:

```markdown
### [Nome_do_Membro]
* **O que fiz ontem?** (ex: Configurei o Repositório e iniciei o Vite PWA)
* **O que planejo fazer hoje?** (ex: Criar as rotas da Tela de Login)
* **Possuo impedimentos?** (ex: Não. / Sim, esperando definição de rotas da API)
```
