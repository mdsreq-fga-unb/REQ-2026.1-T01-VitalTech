# Bem-vindo ao Projeto VitalTech!

**Disciplina de Requisitos de Software - Turma 01 - 2026.1 | Universidade de Brasília (UnB/FGA)**

Este é o site oficial da documentação do software desenvolvido para o **Lar dos Velhinhos Bezerra de Menezes**, uma valiosa Entidade Beneficente de Assistência Social localizada em Sobradinho-DF.

Nosso objetivo central é otimizar, estruturar e digitalizar a rotina diária dos cuidadores e da equipe multidisciplinar por meio de uma solução móvel. Visamos substituir pranchetas físicas e sistemas legados (como o Access) por dados centralizados, garantindo um **monitoramento de excelência, em tempo real**, da saúde, alimentação e bem-estar dos idosos residentes no acolhimento.
